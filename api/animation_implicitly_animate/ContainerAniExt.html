ContainerAniExt extension

为Container组件添加隐式动画效果。

curve 动画曲线，默认为线性。 duration 动画持续时间，默认为300毫秒。 onEnd 动画结束时的回调函数。 width 容器宽度。 height 容器高度。

返回一个AnimatedContainer组件。

on
  • Container

Methods

animate({Curve curve = Curves.linear, Duration duration = const Duration(milliseconds: 300), VoidCallback? onEnd, double? width, double? height}) → AnimatedContainer
