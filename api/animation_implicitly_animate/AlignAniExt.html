AlignAniExt

AlignAniExt extension

为Align组件添加隐式动画效果。

curve 动画曲线，默认为线性。 duration 动画持续时间，默认为300毫秒。 onEnd 动画结束时的回调函数。

返回一个AnimatedAlign组件。

on
  • Align

Methods

animate({Curve curve = Curves.linear, Duration duration = const Duration(milliseconds: 300), VoidCallback? onEnd}) → AnimatedAlign
