为Widget添加大小动画效果。

curve 动画曲线，默认为线性。 alignment 对齐方式，默认为Alignment.center。 duration 动画持续时间，默认为300毫秒。 reverseDuration 反向动画持续时间。 clipBehavior 剪裁行为，默认为Clip.hardEdge。

返回一个AnimatedSize组件。
