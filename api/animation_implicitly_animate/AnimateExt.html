AnimateExt

AnimateExt extension

为Widget添加缩放动画效果。

curve 动画曲线，默认为线性。 duration 动画持续时间，默认为300毫秒。 onEnd 动画结束时的回调函数。 scale 缩放比例。 alignment 对齐方式，默认为Alignment.center。 filterQuality 过滤质量。

返回一个AnimatedScale组件。
